1. Installation

  1. Prerequisites: Ensure you have set up the required database for the system.
  2. Run the Application: Execute the application script to launch the system.

2. Adding a New Member by teacher

  1. Input Member Details: Fill in the member's first name, last name, and address in the provided entry fields.
  2. Add Member: Click the "Add Member" button to register the new member in the system.
  3. Success Message: Upon successful addition, a confirmation message will be displayed.
  4. Failure Message: In case of errors (e.g., incomplete fields), an error message will guide you on the issue.
